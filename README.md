# SSRatingBar
Use this rating bar to show ratings. Really flexible. Generates random colors always. Based on the width.
Pan on the rating bar to set a rating. 

#Usage
`   [self.ratingBar setMaxValue:10];`

   `[self.ratingBar setRating:2];`
